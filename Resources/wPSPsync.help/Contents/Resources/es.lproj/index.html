Ayuda de wPSPsync

wPSPsync analiza una raíz de almacenamiento PSP y una raíz de sincronización, compara carpetas de guardado y copia las partidas más recientes o faltantes al otro lado.

Aviso importante sobre riesgos y garantía

wPSPsync copia, reemplaza y restaura carpetas de guardado. Una raíz elegida incorrectamente, una copia interrumpida, un fallo de almacenamiento, un error de software o una estructura de guardado inesperada pueden causar pérdida de datos o sobrescribir partidas más recientes.

El software se proporciona tal cual, sin garantía de ningún tipo. Las copias automáticas son una medida de seguridad, no una garantía.

Elegir raíces

Elija la carpeta de nivel superior para ambos lados. La raíz de almacenamiento PSP debe contener PSP/SAVEDATA. La raíz de sincronización es la carpeta que contiene o contendrá PSP/SAVEDATA.

Si elige por accidente PSP o SAVEDATA, wPSPsync normaliza la ruta al analizar y sincronizar.

Las raíces seleccionadas se guardan y se restauran entre inicios.

Sincronizar partidas guardadas

Cada fila muestra la fecha de PSP, la fecha de sincronización, el tamaño, el título del juego y el estado de sincronización.

  • Sincronizado: ambos lados tienen la misma fecha del archivo de guardado más reciente.
  • PSP más reciente: la copia de PSP se copiará a la raíz de sincronización.
  • Sincronización más reciente: la copia sincronizada se copiará al almacenamiento PSP.
  • Solo en PSP o solo en sincronización: se creará el lado faltante.

Use las casillas para elegir qué se sincronizará. El botón Sincronizar solo afecta a las filas marcadas.

Copias de seguridad y restauración

Las copias de seguridad están activadas de forma predeterminada. Antes de que wPSPsync escriba un nuevo estado de sincronización, crea una copia zip de la raíz de sincronización actual.

Las copias se guardan en ~/Library/Application Support/wPSPsync/Backups. La app conserva las cinco copias más recientes.

Use Restaurar copia para reemplazar la raíz de sincronización seleccionada con una copia guardada. Cuando las copias están activadas, el estado actual se guarda antes de restaurar.

Desactive Crear copia de seguridad antes de escribir si no desea copias automáticas.

Riesgos y sin garantía

wPSPsync copia, reemplaza y restaura carpetas de guardado. Una raíz elegida incorrectamente, una copia interrumpida, un fallo de almacenamiento, un error de software o una estructura de guardado inesperada pueden causar pérdida de datos o sobrescribir partidas más recientes.

Revise las filas marcadas antes de sincronizar y mantenga copias separadas de las partidas importantes.

Las copias automáticas son una medida de seguridad, no una garantía. Se guardan en el mismo Mac y solo se conservan las cinco copias más recientes.

El software se proporciona tal cual, sin garantía de ningún tipo.

Active Buscar en la API de SerialStation para buscar títulos e imágenes adicionales. La búsqueda de red no cambia los archivos de guardado.

Solución de problemas

Si no se pueden abrir las carpetas, elija de nuevo las raíces desde la barra lateral para que macOS conceda acceso a wPSPsync.

Si una PSP no se detecta automáticamente, elija la raíz de almacenamiento PSP manualmente. La detección automática requiere un volumen montado que contenga PSP/SAVEDATA.

Si existen carpetas SAVEDATA anidadas accidentalmente de una versión anterior, elimínelas manualmente después de confirmar que no contienen partidas necesarias.
